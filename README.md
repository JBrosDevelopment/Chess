# Chess

A simple chess game implemented in C. The game allows two players to play chess against each other on the command line. The game includes basic rules and movement for all chess pieces. It includes advanced features such as castling and promotion, but not  en passant. It can check for check but not checkmate. The game is played on an 8x8 board, and the players take turns moving their pieces. The game ends when one player captures the other player's king or when one player resigns.
